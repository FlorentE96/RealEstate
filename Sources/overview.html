INF01057 - Final project : RealEstate

RealEstate is real estate agent management utility. It allows an agent to manage his performance, clients and to perform sales.

Key Features

  • Independent logging of different agents
  • Evolution of the agent's performance
  • Registration of new agents, clients and property
  • Utility to perform a sale

Abandoned features

  • Encryption of the data and password files
  • Management of rented properties
  • Personalized recommendations based on clients' preferences
  • Distinction between properties for sale and not for sale

Unfinished features

  • Salary level calculation
  • Salary update based on performance
  • Verification of a client's trustability for a sale
